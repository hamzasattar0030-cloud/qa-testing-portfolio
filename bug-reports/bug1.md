# Bug Report

## BUG_001
**Title:** Error message for invalid login is not user-friendly  

**Module:** Login  

**Environment:**  
- Browser: Chrome  
- OS: Windows 11  

**Preconditions:**  
User is on login page  

**Steps to Reproduce:**  
1. Enter valid username  
2. Enter incorrect password  
3. Click login  

**Expected Result:**  
A clear and user-friendly error message explaining the login failure is displayed  

**Actual Result:**  
Generic error message "Username and password do not match" is displayed  

**Severity:** Low  
**Priority:** Low  
**Status:** New
