# BUG01 – Invalid Login Error Message

## Steps to reproduce
1. Open the SauceDemo login page
2. Enter a valid username
3. Enter an invalid password
4. Click the login button

## Expected result
A clear error message should be shown to explain that the login details are incorrect

## Actual result
An error message appears, but this test was logged to document the behaviour and evidence

Medium
