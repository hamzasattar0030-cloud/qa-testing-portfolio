# Test Plan – SauceDemo Login

## Feature being tested
Login page

## Goal
To check that users can log in with correct details and get the right error messages when details are wrong or missing.

## In scope
- Valid login
- Invalid login
- Blank username
- Blank password
- Blank username and password
- Locked out user
- Error message behaviour

## Out of scope
- Cart
- Checkout
- Inventory page
- Performance testing
- Security testing

## Test type
- Functional testing
- Negative testing
- Exploratory testing

## Browser / device
- Chrome
- Laptop

## Deliverables
- Test cases
- Bug reports
